## 4. 형식 세부 이슈 (수정 필요)

- **Alpaca 키 대소문자**: 파일의 키가 `Instruction / Input / Output`으로 **대문자 시작**. Unsloth·HuggingFace 표준 alpaca 매핑은 소문자 `instruction / input / output`을 기대하므로, 기본 템플릿 매핑에서 `KeyError` 발생 가능. → 소문자로 변경하거나 커스텀 매핑 필요.
- **길이**: 최대 총 문자 길이 347자 → `max_seq_length` 이슈 없음.
