## 3. 개선 권고

1. **OCR 재처리(최우선)**: 원문 PDF를 고품질 OCR(또는 텍스트 레이어 추출)로 재변환하여 음절 분리 아티팩트 제거. OCR 품질이 낮으면 하위 지표(유사도·grounding) 전체가 연쇄적으로 저하됨
2. **출력(answer) 재생성**: 정제된 원문 기반으로 output을 재생성해 Semantic Similarity·Grounding 동시 개선
3. **환각 조항 제거**: 원문에 없는 조항 번호 인용 10건 수정 또는 삭제
4. **휴먼 리뷰 체계 도입**: 레코드별 reviewer, review_date, review_result 필드 추가 후 95% 이상 검수 수행
5. **원천 다양화**: 단일 PDF 1종 기반 → 원천 문서 확대로 데이터 대표성 확보

## 4. 측정 방법 및 한계

- OCR 정확도·Semantic Similarity는 원본 PDF 및 임베딩 모델 부재로 **근사(proxy) 측정**임: OCR은 텍스트 아티팩트 비율, 유사도는 문자 n-gram 코사인 사용
- Grounding·Hallucination은 데이터셋에 내장된 메타데이터 값을 그대로 집계
- 중복·Metadata·형식 유효성은 전수(229건) 직접 검사
